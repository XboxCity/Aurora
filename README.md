# Aurora
Aurora mod for work with xboxcity

Релиз модификации Авроры для сайта [xboxcity](https://xboxcity.ru).
В модификации все эндпоинты исправлены на сайт xboxcity.
В том числе авторизация с помощью логина и пароля(в поле профиль-учётная запись Unity)
Сайт поддерживает:
- выбор обложек для игр
- накладывание своего хедера на обложку

если Вы заметили какой то баг на сайте, можете создать тут "проблему"
так же можно писать свои пожелания по функционалу сайта
